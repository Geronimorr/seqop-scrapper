Estabelecer as Diretrizes Técnicas de Projeto de MPD e definir o fluxo de trabalho entre as 
gerências envolvidas.  
 
2. APLICAÇÃO

Executiva de Poços.

Este Padrão define os requisitos de projeto de MPD a serem realizados por 
POCOS/SPO/PEP/PROJ-PERF, sendo usado como referência para o Manual de Projeto de 
MPD. Os requisitos Técnicos estão dispostos no Anexo 1
